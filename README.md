## Why DEBM Outperforms LSTM and GAN (Not Just Statistically)

| Factor | LSTM | GAN | **DEBM (Ours)** |
|--------|------|-----|-----------------|
| Temporal modelling | ✓ LSTM | ✗ MLP only | ✓ BiLSTM encoder |
| Energy-based regularisation | ✗ | ✗ | ✓ E(h,s) via MLP |
| Uncertainty estimation | ✗ | ✗ | ✓ Langevin ensemble |
| Rare event handling | ✗ | Partial | ✓ Dreaming Phase |
| Sentiment injection | ✗ | ✗ | ✓ Concatenated to h |
| Training stability | ✓ Stable | ✗ Mode collapse | ✓ PCD + MSE jointly |
| Over-smoothing | ✓ Common | Variable | ✓ CD loss prevents it |

The DEBM is not just an architectural novelty. The three concrete reasons
it achieves lower RMSE and higher Directional Accuracy in practice:

1. **Bidirectional LSTM encoder** — same temporal power as the LSTM baseline,
   but the latent vector h is then regulated by the energy function.
2. **Contrastive Divergence training** — the energy function discourages
   over-smooth predictions by learning that "average" forecasts have higher
   energy than sharp, realistic ones.
3. **Dreaming Phase** — the model is exposed to synthetic extreme scenarios
   generated by Langevin Dynamics, so it is not surprised by volatility
   spikes or crashes at test time.

## Quick Start

### 1. Install
```bash
pip install -r requirements.txt
```

### 2. (Optional) Set News API key
```bash
export NEWS_API_KEY="your_key_from_newsapi.org"
# Or create a .env file with NEWS_API_KEY=...
```

### 3. Train + evaluate (CLI)
```bash
python pipeline.py --ticker AAPL --epochs 60
# Results saved to /outputs, models to /models
```

### 4. Start the API server
```bash
uvicorn src.api.main:app --host 0.0.0.0 --port 8000 --reload
```

### 5. Open the web UI
Serve `templates/index.html` or open it directly in a browser.
Set `const API = "http://localhost:8000"` in the script tag.

## API Usage

### Train
```bash
curl -X POST http://localhost:8000/train \
     -H "Content-Type: application/json" \
     -d '{"ticker": "AAPL", "force_refresh": false}'
```

### Real-time Prediction
```bash
curl -X POST http://localhost:8000/predict \
     -H "Content-Type: application/json" \
     -d '{"ticker": "AAPL", "use_sentiment": true}'
```

### Response
```json
{
  "ticker": "AAPL",
  "predicted_price": 189.45,
  "confidence_band": {"low": 187.12, "high": 191.78},
  "sentiment_score": 0.142,
  "model": "DreamingAI-DEBM",
  "timestamp": "2025-04-08 12:00:00",
  "last_known_price": 188.20
}
```

## Key Hyperparameters (config.py)

| Parameter | Default | Effect |
|-----------|---------|--------|
| WINDOW_SIZE | 60 | Trading days of look-back |
| LANGEVIN_STEPS | 60 | More steps = better mixing, slower |
| CD_WEIGHT | 0.05 | Higher = stronger energy regularisation |
| DREAM_CYCLES | 2 | More cycles = more synthetic data exposure |
| LSTM_HIDDEN | 128 | Encoder capacity |
| ENERGY_HIDDEN | [256,256,128] | Energy MLP depth |
